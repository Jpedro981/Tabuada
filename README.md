# 📌 Projeto: Tabuada

## 📖 Sobre o Projeto
Este é um projeto simples que gera a tabuada de um número escolhido pelo usuário. Ele pode ser usado para fins educativos ou como exercício de lógica de programação.

## 🚀 Tecnologias Utilizadas
- ✅ [Javascript]
- ✅ [HTML e CSS]
